# Phase 4 — UI Design Contract

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline element spacing |
| sm | 8px | Compact element spacing, form field gaps, badge margin |
| md | 16px | Default element spacing, card internal padding, tab content gap |
| lg | 24px | Section padding within panels, card body padding (upgraded from current 20px) |
| xl | 32px | Layout gaps between panels, dashboard grid gap |
| 2xl | 48px | Page horizontal padding, major section breaks |

Exceptions:
- Touch targets minimum 44px height on all interactive elements (buttons, checkboxes, select triggers) per WCAG 2.1 AA
- Left border indicator on PatientCard stays 4px width (semantic, not spacing)

## Color

The clinical color system from `src/app/globals.css` is authoritative. shadcn CSS variables are remapped to these tokens (D-09). No new colors are introduced.

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | --color-background | #FFFFFF | Page background, panel backgrounds |
| Secondary (30%) | --color-surface | #F8FAFC | Card backgrounds, sidebar, table row alternation |
| Accent (10%) | --color-accent | #0EA5E9 | See reserved list below |
| Border | --color-border | #E2E8F0 | Card borders, panel dividers, table rules |
| Text primary | --color-primary | #0F172A | Body text, headings, primary content |
| Text secondary | --color-secondary | #475569 | Metadata, descriptions, secondary labels |
| Text muted | --color-muted | #94A3B8 | Timestamps, placeholder text, disabled states |
| Accent hover | --color-accent-hover | #0284C7 | Hover state for accent elements |
| Accent foreground | --color-accent-foreground | #FFFFFF | Text on accent-colored backgrounds |
| Destructive | --color-flag-critical | #EF4444 | Destructive actions, critical flags |
| Warning | --color-flag-warning | #F59E0B | Warning flags, escalation state |
| Safe | --color-flag-safe | #10B981 | Resolved flags, approved states, safe indicators |
| Under review | --color-flag-review | #6366F1 | Under-review flag state |

### Accent Reserved For

Accent (#0EA5E9) is used ONLY on these elements:
- Primary action buttons: "Begin Dictation", "Submit Note", "Generate Handoff Report", "Look Up Supplies"
- Active tab indicator (selected tab underline/highlight)
- Focus ring on interactive elements (focus-visible:ring-accent)
- Header bar app title or nav accent element
- Links within report content

Accent is NOT used on: card backgrounds, section headers, decorative elements, borders (except focus rings).

## Realtime Visual Behavior

### Silent Insert Pattern (D-02)

When new data arrives via Supabase Realtime:
- New notes prepend to top of notes list with no animation, no toast, no banner (D-05)
- Flag badges on dashboard cards update in place (D-07)
- Last note timestamp on dashboard cards updates in place (D-07)
- Supply requests and handoff reports appear in their respective tabs silently
- Audit log entries append to Activity tab timeline silently

No visual indication that "something just updated." The data being current IS the demo signal.

### Subscription Lifecycle

- Dashboard (`/`): Global subscription on `notes` table for all patients. Updates flag badges + last note timestamp on PatientCards. Subscribe on mount, unsubscribe on unmount.
- Patient detail (`/patient/[id]`): Per-patient subscriptions on `notes`, `supply_requests`, `handoff_reports`, `audit_log` filtered by `patient_id`. Subscribe on mount, unsubscribe on unmount/navigation (D-04).
- `patient_summaries`: No realtime subscription. Refetched on navigate to patient detail (D-06).

### Keyboard Navigation Requirements (WCAG 2.1 AA)

- All shadcn components ship with Radix keyboard nav -- do not override
- Tab key moves focus through interactive elements in DOM order
- Tabs component: Arrow Left/Right to switch tabs, Tab to enter tab content
- Select component: Arrow Up/Down to navigate options, Enter to select, Escape to close
- AlertDialog: Focus trapped inside when open, Escape to cancel
- Checkbox: Space to toggle

### Focus Indicators

- All interactive elements: `focus-visible:ring-2 focus-visible:ring-accent focus-visible:ring-offset-2`
- Ring color: --color-accent (#0EA5E9)
- Ring offset: 2px against --color-background
- Never suppress focus-visible outlines
